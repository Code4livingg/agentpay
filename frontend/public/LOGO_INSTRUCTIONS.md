# Logo Files Required

Please add the following logo files to this directory:

## 1. logo-mark.png
- Square logo mark (icon only)
- Recommended size: 512x512px or larger
- Transparent background (PNG with alpha channel)
- Should work well at small sizes (36px)

## 2. logo-full.png
- Full horizontal logo (icon + wordmark)
- Recommended size: 1200x300px (4:1 aspect ratio)
- Transparent background (PNG with alpha channel)
- Should be readable at 280px width

## 3. favicon.ico
- Square favicon
- Standard sizes: 16x16, 32x32, 48x48
- ICO format

## Design Guidelines
- Primary color: Purple (#A855F7 / rgb(168, 85, 247))
- Should work on dark backgrounds
- Clean, modern, tech-focused aesthetic
- Consistent with the AgentPay brand

## Current Usage
- Navigation: logo-mark.png at 36px
- Landing hero: logo-full.png at 280px with purple glow effect
- Browser tab: favicon.ico

Once these files are added, the Logo component will automatically display them.
